HOW TO INSTALL

Simply download this 7z file
Extract it using the windows native unzip software (right click file > Extract all)
click into the Gameplay Message router Folder
Copy the 2nd Subfolder
With UE closed, Paste this folder in your Unreal Engine 5.7 plugins Folder (Usually C://Program Files\Epic Games\UE_5.7\Engine\Plugins)
Open UE and go to edit>plugins and search for GameplayMessageSubsystem and verify it's enable.
