HOW TO INSTALL

1. Simply download this 7z file

2. Extract it using the windows native unzip software (right click file > Extract all)
   
3. click into the Gameplay Message router Folder

4. Copy the 2nd Subfolder

5. With UE closed, Paste this folder in your Unreal Engine 5.7 plugins Folder (Usually C://Program Files\Epic Games\UE_5.7\Engine\Plugins)

6. Open UE and go to edit>plugins and search for GameplayMessageSubsystem and verify it's enable.
